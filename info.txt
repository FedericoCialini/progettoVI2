click on rect(cluster): expand the cluster
double-click on rect-shadow(expanded cluster): implode the cluster(and all the inner ones)
zoom-in/out with the mouse wheel
holding click and move mouse for panning
click on the Load graph options for loading examples or upload and load your own graph
Download graph lets you download the .json of the current drawed graph
how to code right your clustered graph .json for the tool:
{ 
    "clusters" : 
    [[
    	[
    		1,
    		2
    	],
    	3,
    	[
    		...
    	],
    	...
    ]],

    // we are interested only in direct graphs
    "edges" : {
        "1" : [2],
        "2" : [5,...],
        ...
    }
}
